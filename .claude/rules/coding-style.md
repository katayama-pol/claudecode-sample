# コーディング規約

このファイルは `.claude/rules/` に置かれた **Rules（ルール）** です。
`paths` フロントマターを指定しているため、Claude が `src/**/*.ts` に一致するファイルを
**読んだとき**に自動でロードされます（条件付きロードのデモ）。

## TypeScript の書き方

- インデントは**スペース2つ**。
- 文末セミコロンは**必ず付ける**。
- `any` は使わない。型は [src/types.ts](../../src/types.ts) に集約する。
- import には拡張子 `.ts` を付ける（例: `import { foo } from "./store.ts"`）。

## データアクセス

- `data/tasks.json` の読み書きは**必ず [src/store.ts](../../src/store.ts) 経由**で行う。
  他のファイルから `fs` で直接読み書きしない。

## ユーザー向けメッセージ

- CLI が出力する**ユーザー向けメッセージはすべて日本語**にする。
- エラーメッセージは `エラー: ` で始める。
